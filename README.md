# Headway Automation — Claude Code Skills

Skills for building reputation and generating leads through forum and Reddit engagement.
Each skill is a prompt file loaded into Claude Code via `/skill-name`.

---

## Skills

| Skill | Trigger | Platform | Goal | Strategy |
|---|---|---|---|---|
| [n8n-forum-reply](skills/n8n-forum-reply/SKILL.md) | `/n8n-reply` | n8n Community Forum | Answer questions, earn points on the Answer & Earn leaderboard, build expert reputation | Short, human reply from personal experience. No AI formatting. No solution mark begging. |
| [reddit-reply](skills/reddit-reply/SKILL.md) | `/reddit-reply` | Reddit | Build authority in automation/SaaS subreddits, drive Upwork profile clicks | Jodianne's Commenting Expert. 90/10 rule. Raw prose, bar test, soft CTA. Never link directly. |

---

## How to install a skill

Copy the `SKILL.md` file into your local Claude Code skills folder:

```
C:\Users\User\.claude\skills\<skill-name>\SKILL.md
```

Then trigger it in any Claude Code session using the trigger command listed above.
